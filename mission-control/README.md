# Mission Control Dashboard

A lightweight static dashboard scaffold for Mike and Hank.

## What it includes

- Responsive single-page dashboard
- Overview stat cards
- Priority queue
- Operations log
- Notes panel with local browser persistence

## Run it

From this directory:

```bash
python3 -m http.server 4173
```

Then open:

<http://localhost:4173>
